# Purpose
The purpose of this repository is to reinforce my understanding of Java programming by solving exercises from multiple platforms and books. By working through these exercises, I aim to improve my coding skills, problem-solving abilities, and familiarity with Java syntax and best practices.

# main
Branch that contains general code for learning and practicing Java programming. This branch includes solutions to exercises from various sources, including online platforms and books.

# exercism
Branch that contains solutions to Java exercises from the Exercism platform.

Exercism is a platform that provides coding exercises to help you improve your programming skills. This repository contains solutions to the Java exercises available on Exercism.

Each exercise is designed to challenge different aspects of Java programming, from basic syntax to more advanced concepts.

My solutions are organized in separate directories for each exercise, and each directory contains the solution code along with any necessary test cases.
